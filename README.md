# Culinary Insight

Web app React + TypeScript para gestión de costos, recetas, inventario, proyecciones y analítica de negocios gastronómicos y hoteleros.

## Ejecutar

```bash
npm install
npm run dev
```

La app funciona en modo demo con `localStorage`. Para conectar Supabase, crea `.env` desde `.env.example`, configura las variables y ejecuta el SQL de `supabase/schema.sql`.

## Flujo incluido

`Recetas` permite crear una receta, elegir ingredientes, definir merma/rendimiento/margen y calcular automáticamente costo unitario, precio sugerido y margen real considerando costos indirectos prorrateados.
